## ðŸ”§ Quick Customization Guide

### Change Primary Color
Edit `/app/globals.css`:
```css
--primary: #3b82f6;  /* Change to your brand color */
```

### Adjust Animation Speed
In component files:
```css
animate-float  /* Change 3s duration */
duration-300   /* Change transition duration */
```

### Modify Gradient
In component files:
```css
from-blue-700 via-blue-500 to-cyan-400  /* Change gradient stops */
```

### Adjust Shadow Intensity
In component classes:
```css
shadow-blue-500/30  /* Change opacity value (30) */
```
